Toggles

Toggle components provide a binary choice with cosmic-themed styling and animations.

Basic Toggles

Basic toggle components for binary choices. Use the label property and bind with [(ngModel)]. 

<nova-toggle
  label="Enable notifications"
  [(ngModel)]="notificationsEnabled">
</nova-toggle>

Disabled Toggles

Toggles can be disabled with the disabled property. 

<nova-toggle
  label="Disabled toggle"
  [disabled]="true"
  [(ngModel)]="toggleState">
</nova-toggle>

Label Position

Toggle labels can be positioned with the labelPosition property. 

<nova-toggle
  label="Label before toggle"
  labelPosition="before"
  [(ngModel)]="labelBeforeToggle">
</nova-toggle>

Toggle with Hints

Add helper text using the hint property. 

<nova-toggle
  label="Enable auto-save"
  hint="Automatically saves your work every 5 minutes"
  [(ngModel)]="autoSaveEnabled">
</nova-toggle>

API Reference

Property Type Default Description
label string '' Label text displayed next to the toggle
checked boolean false Whether the toggle is checked
disabled boolean false Whether the toggle is disabled
required boolean false Whether the toggle is required
hint string undefined Helper text displayed below the toggle
labelPosition 'before' | 'after' 'after' Position of the label relative to the toggle
