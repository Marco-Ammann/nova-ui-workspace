Checkboxes

Nova UI checkboxes provide a cosmic-styled selection control that allows users to select multiple options from a set.

Basic Checkbox

The standard checkbox component with label text.

Default Checkbox

A simple checkbox with a label.

<nova-checkbox
  label="Subscribe to newsletter"
  [(ngModel)]="newsletter">
</nova-checkbox>

With Hint

Checkbox with additional hint text below it.

<nova-checkbox
  label="Enable notifications"
  hint="You'll receive updates about new features"
  [(ngModel)]="notifications">
</nova-checkbox>

Checkbox States

Various states for checkbox components.

Checked

A checkbox in the checked state.

<nova-checkbox
  label="Checked checkbox"
  [(ngModel)]="checkedState">
</nova-checkbox>

// Component:
checkedState = true;

Indeterminate

A checkbox in the indeterminate state, useful for tri-state selections.

<nova-checkbox
  label="Indeterminate checkbox"
  [indeterminate]="isIndeterminate"
  [(ngModel)]="indeterminateState">
</nova-checkbox>

// Component:
isIndeterminate = true;
indeterminateState = false;

Required

A checkbox marked as required with an asterisk.

<nova-checkbox
  label="I agree to the terms and conditions"
  [required]="true"
  [(ngModel)]="termsAgreed">
</nova-checkbox>

Disabled Unchecked

A disabled checkbox in the unchecked state.

<nova-checkbox
  label="Disabled unchecked checkbox"
  [disabled]="true"
  [(ngModel)]="disabledUnchecked">
</nova-checkbox>

// Component:
disabledUnchecked = false;

Disabled Checked

A disabled checkbox in the checked state.

<nova-checkbox
  label="Disabled checked checkbox"
  [disabled]="true"
  [(ngModel)]="disabledChecked">

// Component:

Checkbox Group Example

A group of related checkboxes for selecting multiple options.

Notification Preferences

// Component code
notificationPrefs = {
  email: false,
  push: false,
  sms: false,
  inApp: false
};
allNotificationsSelected = false;
someNotificationsSelected = false;

updateParentCheckbox() {
  const values = Object.values(this.notificationPrefs);
  this.allNotificationsSelected = values.every(v => v);
  this.someNotificationsSelected = values.some(v => v) && !this.allNotificationsSelected;
}

toggleAllNotifications() {
  Object.keys(this.notificationPrefs).forEach(key => {
    this.notificationPrefs[key] = this.allNotificationsSelected;
  });
  this.someNotificationsSelected = false;
}

Forms and Validation

The checkbox component works with Angular's form validation system.

Reactive Form Example

Sign Up 
// Component code
import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({...})
export class YourComponent {
  signupForm: FormGroup;
  
  constructor(private fb: FormBuilder) {
    this.signupForm = this.fb.group({
      email: ['', [Validators.required, Validators.email]],
      subscribe: [false],
      agreeTerms: [false, Validators.requiredTrue]
    });
  }
  
  onSubmit() {
    if (this.signupForm.valid) {
      // Handle form submission
      console.log(this.signupForm.value);
    }
  }
}

API Reference

The complete API reference for the NovaCheckboxComponent.

Selector

nova-checkbox

Inputs

Name Type Default Description
label string '' The text label for the checkbox.
disabled boolean false Whether the checkbox is disabled.
required boolean false Whether the checkbox is required.
indeterminate boolean false Whether the checkbox is in an indeterminate state.
name string undefined The name attribute for the checkbox input.
value any undefined The value attribute for the checkbox input.
hint string undefined Helper text displayed below the checkbox.

Outputs

Name Type Description
change EventEmitter<boolean> Emitted when the checkbox is toggled, with the new state.

NgModel Support

The checkbox component implements ControlValueAccessor, allowing it to work with ngModel and reactive forms.

Styling

Customize the checkbox component with CSS variables and custom styles.

CSS Variables

The checkbox component uses the following CSS variables from the theme system:

Variable Purpose
--nova-primary Color for the checkbox when checked
--nova-primary-rgb RGB values for the primary color, used for glow effects
--nova-on-surface Color for the label text
--nova-surface-alt Background color for the unchecked checkbox
--nova-error Color for the required marker

Custom CSS

You can customize checkbox appearance with CSS:

/* Custom checkbox styling */ nova-checkbox.custom-checkbox .nova-checkbox__box { border-radius: 0.125rem; border-width: 2px; } /* Custom label styling */ nova-checkbox.custom-checkbox .nova-checkbox__label { font-weight: 500; } /* Custom checked state styling */ nova-checkbox.custom-checkbox.nova-checkbox--checked .nova-checkbox__box { background-color: var(--nova-accent2); border-color: var(--nova-accent2); }
