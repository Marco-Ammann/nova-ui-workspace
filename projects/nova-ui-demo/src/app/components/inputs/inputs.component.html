Inputs

Nova UI provides beautifully styled input components for gathering user data with cosmic-themed visual effects.

Basic Input

The standard text input component with a label, placeholder, and optional hint.

Default Input

A basic text input with a label and placeholder. 

<nova-input
  label="Username"
  placeholder="Enter your username"
  [(ngModel)]="username">
</nova-input>

With Hint

Input with additional hint text below it. 

<nova-input
  label="Username"
  placeholder="Enter your username"
  hint="Choose a name between 3-20 characters"
  [(ngModel)]="usernameWithHint">
</nova-input>

Input States

Various states for input components.

Required

Indicates a required field with an asterisk. 

<nova-input
  label="Username"
  placeholder="Enter your username"
  [required]="true"
  [(ngModel)]="requiredUsername">
</nova-input>

Disabled

A non-interactive, disabled input.

<nova-input
  label="Username"
  placeholder="Enter your username"
  [disabled]="true"
  [(ngModel)]="disabledUsername">
</nova-input>

Read-only

Non-editable input for display purposes. 

<nova-input
  label="Username"
  [readonly]="true"
  [(ngModel)]="readonlyUsername">
</nova-input>

Forms and Validation

The input component works with Angular's form validation system to display error states.

Reactive Form Example

Login 
// login-form.component.ts
  import { Component } from '@angular/core';
  import { FormBuilder, FormGroup, Validators } from '@angular/forms';
  
  @Component({
    selector: 'app-login-form',
    template: `
      

        
        
  
        
        
  
        
          Login
        
      

    `
  })
  export class LoginFormComponent {
    loginForm: FormGroup;
    
    constructor(private fb: FormBuilder) {
      this.loginForm = this.fb.group({
        email: ['', [Validators.required, Validators.email]],
        password: ['', [Validators.required, Validators.minLength(8)]]
      });
    }
    
    onSubmit() {
      if (this.loginForm.valid) {
        // Handle form submission
        console.log(this.loginForm.value);
      }
    }
  }

API Reference

The complete API reference for the NovaInputComponent.

Selector

nova-input

Inputs

Name Type Default Description
label string | undefined undefined The label text displayed above the input.
placeholder string '' Placeholder text shown when the input is empty.
type string 'text' The HTML input type ('text', 'email', 'password', etc.).
disabled boolean false Whether the input is disabled.
readonly boolean false Whether the input is read-only.
required boolean false Whether the input is required.
hint string | undefined undefined Helper text displayed below the input.
iconLeft string | undefined undefined Icon to display on the left side of the input.
iconRight string | undefined undefined Icon to display on the right side of the input.
autocomplete string | undefined undefined Value for the HTML autocomplete attribute.

Outputs

Name Type Description
blur EventEmitter<FocusEvent> Emitted when the input loses focus.
focus EventEmitter<FocusEvent> Emitted when the input receives focus.

NgModel Support

The input component implements ControlValueAccessor, allowing it to work with ngModel and reactive forms.

Styling

Customize the input component with CSS variables and custom styles.

CSS Variables

The input component uses the following CSS variables from the theme system:

Variable Purpose
--nova-primary Used for focus states and active label color
--nova-error Used for error states and validation messages
--nova-on-surface Text color for input and label
--nova-surface-alt Background color for the input field
--nova-font-family Font family for input text

Custom CSS

You can customize input appearance with CSS:

/* Custom input styling */ nova-input.custom-input .nova-input__field { border-radius: 2rem; background-color: rgba(var(--nova-surface-alt-rgb), 0.3); border-width: 2px; } /* Custom label styling */ nova-input.custom-input .nova-input__label { font-size: 0.8rem; text-transform: uppercase; letter-spacing: 0.05em; } /* Custom focus effect */ nova-input.custom-input .nova-input__focus-effect::before { background: radial-gradient( circle at center, rgba(var(--nova-secondary-rgb), 0.2), transparent 70% ); }
