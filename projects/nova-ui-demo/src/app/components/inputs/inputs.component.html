Inputs

Nova UI provides beautifully styled input components with cosmic-themed visual effects.

Basic Input

The standard text input component with a label, placeholder, and optional hint.

Default Input

<nova-input
  label="Username"
  placeholder="Enter your username"
  [(ngModel)]="username">
</nova-input>

With Hint

<nova-input
  label="Username"
  placeholder="Enter your username"
  hint="Choose a name between 3-20 characters"
  [(ngModel)]="usernameWithHint">
</nova-input>

Input Types

Different input types for various data needs.

Email Input

<nova-input
  label="Email Address"
  placeholder="name@example.com"
  type="email"
  [(ngModel)]="email">
</nova-input>

Password Input

<nova-input
  label="Password"
  placeholder="Enter your password"
  type="password"
  [(ngModel)]="password">
</nova-input>

Number Input

<nova-input
  label="Age"
  placeholder="Enter your age"
  type="number"
  [(ngModel)]="age">
</nova-input>

Input States

Various states for input components.

Required

<nova-input
  label="Username"
  placeholder="Enter your username"
  [required]="true"
  [(ngModel)]="requiredUsername">
</nova-input>

Disabled

<nova-input
  label="Username"
  placeholder="Enter your username"
  [disabled]="true"
  [(ngModel)]="disabledUsername">
</nova-input>

Read-only

<nova-input
  label="Username"
  [readonly]="true"
  [(ngModel)]="readonlyUsername">
</nova-input>

Forms and Validation

The input component works with Angular's form validation system.

Reactive Form Example

Login 
// Component code
this.loginForm = this.fb.group({
  email: ['', [Validators.required, Validators.email]],
  password: ['', [Validators.required, Validators.minLength(8)]]
});

onSubmit() {
  if (this.loginForm.valid) {
    // Handle form submission
    console.log(this.loginForm.value);
  }
}

API Reference

Property Type Default Description
label string undefined The label text displayed above the input.
placeholder string '' Placeholder text shown when the input is empty.
type string 'text' The HTML input type ('text', 'email', 'password', etc.).
disabled boolean false Whether the input is disabled.
readonly boolean false Whether the input is read-only.
required boolean false Whether the input is required.
hint string undefined Helper text displayed below the input.
iconLeft string undefined Icon to display on the left side of the input.
iconRight string undefined Icon to display on the right side of the input.
