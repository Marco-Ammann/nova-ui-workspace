Buttons

Nova UI buttons are interactive elements that trigger actions or navigation.

Button Appearances

The appearance property defines the visual style of the button.

Basic appearance="basic"
Raised appearance="raised"
Stroked appearance="stroked"
Flat appearance="flat"
appearance="icon" icon="★"
appearance="fab" icon="+"
appearance="mini-fab" icon="-"
Extended appearance="extended-fab" icon="→"

Button Variants

The variant property defines the semantic meaning of the button.

Basic variant="basic"
CTA variant="cta"
Link variant="link"

Button Sizes

The size property adjusts the size of the button.

Small size="small"
Medium size="medium"
Large size="large"

Button States

Buttons can be in normal, disabled, or loading states.

Normal <nova-button>
Disabled [disabled]="true"
Loading [loading]="true"

Common Use Cases

Cancel Save Draft Submit 
<nova-button variant="link">Cancel</nova-button>
<nova-button>Save Draft</nova-button>
<nova-button variant="cta">Submit</nova-button>
