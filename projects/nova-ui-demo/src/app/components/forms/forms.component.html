Form Components

Nova UI provides form components with cosmic-themed styling for collecting user input. These components work with both template-driven and reactive forms in Angular.

Overview

Nova UI form components are designed to provide a consistent, accessible, and visually appealing way to collect user input. Each component features space-themed styling that adapts to your selected theme, with subtle animations and cosmic effects.

Form Components Guidelines

Input Types

NovaInputComponent supports all HTML input types:

  • text (default)
  • email
  • password
  • number
  • tel
  • url
  • search
  • date
  • time
  • and more...

Validation

All form components work with Angular's form validation:

  • Required fields
  • Minimum/maximum length
  • Pattern validation
  • Custom validators
  • Error messages

Accessibility

Form components are built with accessibility in mind:

  • Proper labeling
  • Keyboard navigation
  • Screen reader support
  • Focus management
  • High contrast modes

Best Practices

Follow these guidelines for the best user experience:

  • Use clear, concise labels
  • Provide helpful hints when needed
  • Group related fields together
  • Use appropriate input types
  • Show validation errors inline
