Using Themes in Your Application

1. Set Theme Programmatically

Use the NovaThemeService to set themes in your components:

constructor(private themeService: NovaThemeService) {}

// Set theme base (color palette)
setTheme(theme: NovaThemeBase): void {
  this.themeService.setThemeBase(theme);
}

// Toggle between light and dark mode
toggleMode(): void {
  this.themeService.toggleThemeMode();
}

2. Add Theme Toggler Component

Use the built-in theme toggler component in your application:

<nova-theme-toggler></nova-theme-toggler>

The toggler provides a user-friendly interface for changing themes and dark/light modes.

3. Create Custom Theme

You can create your own custom theme by extending Nova UI themes:

/* Custom theme */
:root {
  --nova-primary: #your-color;
  --nova-secondary: #your-color;
  --nova-accent1: #your-color;
  /* Add all required CSS variables */
}
