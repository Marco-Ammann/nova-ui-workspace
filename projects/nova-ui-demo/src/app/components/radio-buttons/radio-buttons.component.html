API Reference

Complete reference of all properties and events available for the NovaRadioButtonComponent.

Properties

Property Type Default Description
label string '' Label text displayed next to the radio button
value any undefined The value of the radio button
name string '' The name attribute of the radio button (group identifier)
checked boolean false Whether the radio button is checked
disabled boolean false Whether the radio button is disabled
required boolean false Whether the radio button is required
hint string undefined Helper text displayed below the radio button

Events

Event Type Description
change EventEmitter<any> Emitted when the radio button is selected

Customization

The NovaRadioButtonComponent can be customized using CSS variables and custom styles.

Using CSS Variables

You can customize the appearance of radio buttons by overriding the following CSS variables:

:root { /* Radio button colors */ --nova-radio-unchecked-color: rgba(var(--nova-on-surface-rgb), 0.3); --nova-radio-checked-color: var(--nova-primary); --nova-radio-hover-color: var(--nova-primary); /* Radio button sizing */ --nova-radio-size: 1.25rem; --nova-radio-dot-size: 0.625rem; /* Radio button animation */ --nova-radio-animation-duration: 0.2s; }

Extending with Global Styles

You can also apply global styles to all radio buttons in your application:

/* In your global styles.scss */ .nova-radio__label { /* Custom styles for the label */ font-weight: 500; } .nova-radio__dot { /* Custom styles for the radio dot */ background-color: var(--nova-accent1); }
